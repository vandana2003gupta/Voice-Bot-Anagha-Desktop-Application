# 🎙 Voice-Bot-Anagha

A Python-based desktop voice assistant that can respond to voice commands using text-to-speech and speech recognition. It integrates features like weather updates using the OpenWeatherMap API, Wikipedia search, jokes, and more.

# Features

🎤 Voice recognition using speech_recognition
🗣️ Text-to-speech conversion with pyttsx3
⛅ Real-time weather updates using OpenWeatherMap API
📚 Wikipedia search
😂 Fetch random jokes using pyjokes
📅 Tells the current date and time
🔁 Interactive voice responses

# Project Structure
Voice Bot Anagha/
├── build/
├── dist/
│   └── Voice Bot Spyder.exe      # Standalone executable
├── Scripts/                      # Python scripts and configs
├── Voice Bot Spyder.py          # Main Python script
├── Voice Bot Spyder.spec        # PyInstaller spec file
├── requirements.txt             # Required Python packages
└── .venv/                        # Virtual environment

# Installation

Clone the repository:
git clone https://github.com/your-username/voice-bot-anagha.git
cd voice-bot-anagha

Create a virtual environment:
python -m venv venv
.\venv\Scripts\activate

Install dependencies:
pip install -r requirements.txt
